# 操作系统

## LINUX

### 常见命令

- 目录切换

  ```
  cd
  ```

- 目录操作

  ```
  mkdir
  ls
  find
  mv
  cp
  rm
  ```

- 操作文件

  ```
  touch
  cat
  vim
  ```

- 用户管理

  ```
  useradd
  userdel
  usermod
  passwd
  ```

- 权限管理

  ```
  chmod
  rwx
  u g r
  ```

- 其他常用命令

  ```
  pwd
  grep
  ps
  kill
  shutdown
  reboot
  ```
